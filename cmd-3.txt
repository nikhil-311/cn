STEP 6 Ś Create mapper.py
Run:
nano mapper.py
Paste:
#!/usr/bin/env python3

import sys

for line in sys.stdin:
    line = line.strip()

    if line:
        name, marks = line.split(',')

        print(f"{name}\t{marks}")
Save:

* Enter

STEP 7 Ś Create reducer.py
Run:
nano reducer.py
Paste:
#!/usr/bin/env python3

import sys

for line in sys.stdin:
    name, marks = line.strip().split('\t')

    marks = int(marks)

    if marks >= 90:
        grade = 'A'

    elif marks >= 80:
        grade = 'B'

    elif marks >= 70:
        grade = 'C'

    elif marks >= 60:
        grade = 'D'

    else:
        grade = 'F'

    print(f"{name}\t{grade}")
Save:
* CTRL + O 
* Enter 
* CTRL + X 

STEP 8 Ś Give Execution Permission
Run:
chmod +x mapper.py
chmod +x reducer.py

STEP 9 Ś Create HDFS Input Folder
Run:
hdfs dfs -mkdir /grades_input

STEP 10 Ś Upload Input File to HDFS
Run:
hdfs dfs -put students.txt /grades_input

STEP 11 Ś Verify File Uploaded
Run:
hdfs dfs -ls /grades_input
Expected:
students.txt

STEP 12 Ś Run Hadoop MapReduce Job
MOST IMPORTANT COMMAND:
hadoop jar /usr/lib/hadoop-mapreduce/hadoop-streaming.jar \
-files mapper.py,reducer.py \
-mapper mapper.py \
-reducer reducer.py \
-input /grades_input/students.txt \
-output /grades_output
Wait until Hadoop finishes.

STEP 13 Ś View Final Output
Run:
hdfs dfs -cat /grades_output/part-00000

EXPECTED OUTPUT
From your manual: 
Alice  B
Bob    D
Charlie A
David  C
Eva    F

IMPORTANT ERROR FIX
If You Get:
Output directory already exists
Run:
hdfs dfs -rm -r /grades_output
Then rerun STEP 12.

Important Viva Questions
What does Mapper do?
Mapper reads student names and marks and emits key-value pairs.

What does Reducer do?
Reducer assigns grades based on marks using conditional logic.

Why use Hadoop?
Hadoop enables distributed processing of large datasets.

What is HDFS?
HDFS is Hadoop Distributed File System used to store big data.
