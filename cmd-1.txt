STEP 7 Ś Create reducer.py
Run:
nano reducer.py
Paste:
#!/usr/bin/env python3

import sys

current_word = None
current_count = 0

for line in sys.stdin:
    line = line.strip()

    word, count = line.split('\t')
    count = int(count)

    if current_word == word:
        current_count += count
    else:
        if current_word:
            print(current_word + "\t" + str(current_count))

        current_word = word
        current_count = count

if current_word:
    print(current_word + "\t" + str(current_count))

STEP 9 Ś Create HDFS Input Folder

STEP 12 Ś Run Hadoop MapReduce Job
MOST IMPORTANT COMMAND:
hadoop jar /usr/lib/hadoop-mapreduce/hadoop-streaming.jar \
-files mapper.py,reducer.py \
-mapper mapper.py \
-reducer reducer.py \
-input /input/input.txt \
-output /output
Wait until Hadoop finishes processing.

STEP 13 Ś View Final Output

Important Viva Questions
What is MapReduce?
MapReduce is a programming model used for distributed processing of large datasets.

What does Mapper do?
Mapper converts input data into key-value pairs.

What does Reducer do?
Reducer aggregates intermediate key-value pairs.

What is HDFS?
HDFS is Hadoop Distributed File System used for storing big data.
